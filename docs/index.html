Best Country for Immigrants (English-speaking focus)

A reproducible Python pipeline that downloads open global indicators, normalizes them to 0–1, computes factor scores, and produces a weighted country ranking with an interactive world map.

Economy Safety Healthcare Education & Innovation Environment Infrastructure Governance Migration friendliness
Open map in full screen Download ranking CSV View source code

What this score means

  • Each metric is normalized to 0–1 (1 = better).
  • Factor scores = average of assigned metrics.
  • Final score = weighted sum of factors.

Data sources

  • World Bank indicators
  • Our World in Data (OWID)
  • ND-GAIN climate resilience indices
