# Best Country for Immigrants (Geospatial Scoring Framework)

This project builds a Python-based framework to score countries (0–1) using open indicators:
- Economy
- Safety
- Healthcare
- Education/Innovation
- Environment/Climate (incl. ND-GAIN)
- Infrastructure
- Governance
- Migration friendliness
